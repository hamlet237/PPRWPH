Whitepaper Manager - WPH is a robust and adaptable plugin intended to assist users in the creation, management, and presentation of comprehensive project whitepapers.

== Description ==

**Whitepaper Manager - WPH** is a powerful and flexible plugin designed to help you create, manage, and display detailed project whitepapers. Whether you're documenting internal projects or presenting project details to stakeholders, this plugin streamlines the process of building structured, professional whitepapers within your WordPress environment.

With an intuitive interface, you can break down projects into sections, manage revisions, and share completed documents with your team or the public. It's an essential tool for **project managers**, **developers**, **marketers**, and anyone needing to organize project details.

Fully integrated with Gutenberg edition builder, it will allow you to improve your Whitepapers.

**Features:**
* Create structured project whitepapers.
* Manage multiple whitepaper sections from one dashboard.
* Support for project sections, ordering, edition, hierarchy and removal.
* Version control for whitepaper revisions.
* Gutenberg editor full integration.

Whitepaper Manager - WPH helps bring transparency to your project process and provides a professional format for communicating complex information.


== Installation ==

1. Upload the `pprwph` folder to the `/wp-content/plugins/` directory or install the plugin via the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create your first Whitepaper section or navigate to "Whitepaper Sections" in your WordPress admin dashboard.
4. Start creating and managing your whitepapers by adding project details and publishing them.


== Frequently Asked Questions ==

= How do I create a whitepaper? =
After activating the plugin, go to the "Whitepaper Manager" section in your dashboard. Click "Add New Whitepaper" and fill in the necessary details for your project. Once completed, publish your whitepaper, and use the shortcode provided to display it.

= Is this plugin compatible with all themes? =
Whitepaper Manager - WPH is designed to work with most modern WordPress themes. However, if you encounter any compatibility issues, feel free to reach out for support.

= Can I manage revisions of my whitepapers? =
Absolutely. The plugin includes version control based on WP revisions system, allowing you to save multiple versions of your whitepapers, track changes, and restore previous versions if needed.
